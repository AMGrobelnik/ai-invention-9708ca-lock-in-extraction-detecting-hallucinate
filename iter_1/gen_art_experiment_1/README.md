## Full Summary

Implemented the complete Lock-In Extraction pipeline for hallucination detection using Hadamard-multiplexed phi-coefficient computation. The method builds K document variants via span negation guided by a Hadamard design matrix, extracts atomic facts from each variant using a frozen-schema LLM canonicalization pass, computes phi-coefficients (Pearson correlation between atom presence vectors and channel design vectors), and uses the max-phi score as a hallucination signal.

Five baselines implemented: (1) embedding cosine similarity via all-MiniLM-L6-v2, (2) LLM self-judge (yes/no grounding query), (3) verbatim LCS ratio, (4) self-consistency (5-run re-extraction frequency), (5) LOO single-span negation. Prerequisite gates T1 (cross-variant canonicalization agreement) and T2 (null-document prior subtraction) are computed and reported.

Evaluated on three datasets: multinli (80 NLI premise/hypothesis pairs, entailment=faithful, contradiction=hallucinated), factscore (344 wiki_bio_gpt3_hallucination atoms from 40 topics with per-sentence annotations), summac (60 SNLI proxy examples). Total: 484 atom-level labels.

Results (AUROC): Lock-in achieves 0.528/0.549/0.700 on NLI/FactScore/SummaC. LLM-judge baseline achieves 0.763/0.631/0.933. Embedding baseline achieves 0.552/0.643/0.944. SNR/K ablation over K={4,8,16} and negation-vs-deletion ablation also computed. Calibration (Platt + isotonic) with ECE reported. Multi-hop ProofWriter evaluation included.
